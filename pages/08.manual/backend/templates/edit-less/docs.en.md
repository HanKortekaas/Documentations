### Custom Less

* Go to **_Backend -> Components -> Kunena Forum -> Templates_**
* Mark Template
* Click **Edit LESS**
* Mark `custom.less`
* Click **Edit LESS**
* Enter your Code
* Save & Close
* Empty the Cache

>>>>> The `custom.less` will created automatically, if you edit the first time in the backend the LESS.
